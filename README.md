# Oriental Fashion Ltd. — Garment Manufacturer & Exporter Portal

An exceptionally professional B2B industrial website and inquiry portal for **Oriental Fashion Ltd.**, a premier ready-made garments (RMG) exporter based in Bangladesh since 2011.

## Tech Stack

- **Framework:** TanStack Start (React 19)
- **Styling:** Tailwind CSS 4 (Refined Industrial/B2B Dark Aesthetic)
- **Routing:** TanStack Router (File-based)
- **Spokesperson AI:** Interactive Corporate Assistant (customized with PDF industrial data)
- **B2B Intake Channel:** Fully integrated with Netlify Forms
- **Media System:** 14 high-performance optimized production assets showcasing real factory floor operations

## Features

- **Facility Floor-by-Floor Interactive Layout:** Highlights the 6 floors of manufacturing at Savar, Savar Dhaka (Store, Finishing, Sewing, CAD/Cutting, Offices, and Dining).
- **320-Machine Technical Matrix:** Live dashboard breakdown of factory capacity (auto plain stitchers, flatlocks, overlocks, metal detector units, boilers, and 100 KV generators).
- **Technical Garments Showcase:** Detailed specifications of manufactured garments (Casual/Fancy Shirts, washed Denim Jackets, Fleece jackets, and Child-safe Kids/Ladies bonded apparel) outlining monthly capacity, standard stitches per inch, and materials.
- **Factory Media Gallery:** 14 beautifully optimized visual cards illustrating sewing divisions, pressing lines, fabric stores, and automated CAD designs.
- **AJAX Netlify Forms B2B Intake:** Built-in form with bot spam honey-pot, custom corporate fields (Name, Corporate Email, Brand Name, and Garment Specifications).
- **OF Advisor AI:** Customized chat assistant designed to answer real inquiries regarding factory specifications, machinery lists, building layout, safety fire-pumps, and direct contacts.

## How to Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server using Netlify CLI:
   ```bash
   netlify dev --port 8889
   ```

3. Open your browser at `http://localhost:8889`.
